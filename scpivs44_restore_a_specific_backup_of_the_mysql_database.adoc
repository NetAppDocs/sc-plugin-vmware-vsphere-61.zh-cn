= 恢复 MySQL 数据库的特定备份
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以使用维护控制台SnapCenter Plug-in for VMware vSphereMySQL 数据库（也称为 NSM 数据库）的特定备份。

.步骤
. 打开维护控制台窗口。
+
link:scpivs44_access_the_maintenance_console.html["访问维护控制台"^] 。

. 从主菜单中，输入选项*1)应用程序配置*。
. 从应用程序配置菜单中，输入选项*6) MySQL 备份和恢复*。
. 从 MySQL 备份和恢复配置菜单中，输入选项 *2) 列出 MySQL 备份*，然后记下要恢复的备份。
. 从 MySQL 备份和恢复配置菜单中，输入选项 *4) 恢复 MySQL 备份*。
. 在提示“使用最近的备份恢复”时输入 *n*。
. 在提示“要从中恢复的备份”时输入备份名称，然后按 *Enter*。
+
选定的备份 MySQL 数据库将恢复到其原始位置。
