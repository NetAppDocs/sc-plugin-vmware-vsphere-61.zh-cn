= 创建资源组
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
按照以下步骤创建资源组。

. 在 SCV 插件的左侧导航窗格中，选择 *资源组*，然后选择 *创建*。
. 在创建资源组向导的每个页面上输入所需的信息，选择要包含在资源组中的虚拟机和数据存储，然后选择要应用于资源组的备份策略。添加远程二级保护详细信息并指定备份计划。


备份按照为资源组配置的备份策略中指定的方式执行。

您可以从“资源组”页面选择以下选项来按需执行备份image:scpivs44_image38.png["播放图标"]*立即运行*。
