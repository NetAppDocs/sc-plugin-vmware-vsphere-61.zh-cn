= 修改配置的存储超时
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
即使备份在过去成功运行，SnapCenter Plug-in for VMware vSphere必须等待存储系统超过配置的超时期限期间，备份可能会开始失败。如果出现这种情况，您可以增加配置的超时时间。

您可能会遇到错误 `Unable to discover resources on SCV: Unable to get storage details for datastore <xxx>…`

.步骤
. 在 SCV 插件的左侧导航窗格中，选择 *存储系统*。
. 在存储系统页面上，选择要修改的存储系统，然后选择*编辑*。
. 在超时字段中，增加秒数。
+

TIP: 对于大型环境，建议使用 180 秒。
