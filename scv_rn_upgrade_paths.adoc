= 升级路径
:allow-uri-read: 


[role="lead"]
您可以升级到的适用SnapCenter Plug-in for VMware vSphere版本取决于您当前运行的版本。

[NOTE]
====
仅 VMware vCenter Server 7 Update 1 及更高版本支持升级到SnapCenter Plug-in for VMware vSphere。对于版本 7 更新 1 之前的 VMware vCenter 服务器，您应该继续使用 SCV 4.7。

====
[cols="50%,50%"]
|===
| 如果您使用的是 SCV 版本…… | 您可以直接将 SCV 升级到... 


 a| 
SCV 6.0
 a| 
SCV 6.1



 a| 
SCV 5.0
 a| 
SCV 6.0 和 SCV 6.1



 a| 
SCV 4.9
 a| 
SCV 5.0 和 SCV 6.0



 a| 
SCV 4.8
 a| 
SCV 4.9 和 SCV 5.0



 a| 
SCV 4.7
 a| 
SCV 4.8 和 SCV 4.9

|===

SnapCenter 6.0

 a|

 a|

 a|

 a| 
SCV 4.7
 a| 
SCV 4.8

|===
